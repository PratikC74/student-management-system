# 🎓 Student Management System (CLI-Based)

## 📌 Overview
The Student Management System is a command-line application built using Python that allows users to manage student records efficiently. It supports adding, updating, deleting, and viewing student data with persistent storage using JSON.

---

## 🚀 Features
- ➕ Add new student (ID, Name, Grade)
- ✏️ Update existing student details
- ❌ Delete student records
- 📋 View all students in formatted output
- 🆔 Unique Student ID validation
- 💾 Data persistence using JSON file

---

## 🛠️ Tech Stack
- **Language:** Python 🐍  
- **Concepts Used:**
  - Object-Oriented Programming (OOP)
  - File Handling (JSON)
  - CLI (Command Line Interface)
